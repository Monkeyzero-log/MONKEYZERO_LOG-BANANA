# 文字劇場 Textual Theatre

「字越大越重、換行是重力、標點有質量。」

文字物理學（Textual Physics Codex v12.1）的作品櫃：
所有 demo 都是零依賴的單檔 HTML，打開即玩。

- `index.html` — 作品櫃（分類展示全部作品）
- `demos/` — 27 件作品
- `docs/` — 機制總目錄 v1.0、設計筆記 v0.1

## 本地預覽
直接雙擊 `index.html` 即可。

## 加新作品
把 HTML 放進 `demos/`，在 `index.html` 的 `DEMOS` 陣列加一行。

（除引擎驗收外，各作品文本均為佔位示意，替換資料表即為正式版。）
