# 文字物理學 · 雙館聯展

Sheep & Architect 的 HTML 文字創作實驗場——甲乙兩館共三十六件。
（甲乙館為兩個平行工作對話的成果；同題作品在目錄中相鄰排列，可直接對照。）

## 開啟 GitHub Pages（三步）

1. 把本資料夾的**所有檔案**上傳到 repo 根目錄（不要包在子資料夾）
2. **Settings → Pages → Source**：Deploy from a branch → `main` → `/ (root)` → Save
3. 一兩分鐘後，`https://你的帳號.github.io/repo名稱/` 就是總目錄頁

## 注意事項

- **雙窗劇場** 依賴分頁通訊，file:// 可能失效——放上 Pages 就正常
- **視窗即牢籠、心跳** 桌面體驗最佳；**循環** 使用 localStorage 記憶輪數
- 各作品文本多為**佔位示意**，正式發表前請替換定稿
- 《麵包屑》《黑白棋》含作者本人的同人文文本，repo 設 public 前請自行斟酌
