## 記錄方式

- 原始檔（PSD、CLIP 等）不入庫，只放輸出的 JPG 或 PNG，寬約 2000px、品質 80%
- 檔名以日期開頭：`2026-08-03_box-rotation_01.jpg`
- Commit 訊息寫明內容：`透視課 第2堂作業：兩點透視街景`
- 每三個月挑一張舊圖重畫，與原圖並排收進「精選圖版」
