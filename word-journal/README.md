# 文集

猩其一（gorillaone）的文字記錄——原創、同人與研究，自二〇二六年八月起。

繪圖姊妹冊：[習作輯](../art-journal/)

## 結構

| 部 | 內容 | 頁面 |
|------|------|------|
| 卷一 | 原創 | `original.html` |
| 卷二 | 同人（各篇附原作聲明） | `fanwork.html` |
| 研究 | 技法分析與閱讀筆記 | `research.html` |

正式篇章在 `pieces/`，草稿在 `drafts/`（不會顯示於網站）。

## 新增一篇

1. 複製 `pieces/_template-piece.html` 改名（例：`pieces/2026-09_title.html`）
2. 照檔內註解填標題、日期、字數與正文
3. 到所屬的部（original / fanwork / research.html）篇目加一列
4. `git add .` → `commit` → `push`

字數統計：VS Code 可裝擴充功能「chars-counter」，或貼到任何線上字數工具數一次，手動填進篇目與篇頭。
