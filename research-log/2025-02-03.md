# 方塊座標系統

------


所有方塊的理解只需要記住一個東西：

> **你在看一個正方體，它在哪裡轉、轉了多少 → 決定你看到什麼形狀**

不用記角度，不用記cos，只用座標數值。

------

## 一、座標定義

### 原點 = 正面朝你、平視的方塊

```
(0, 0) = 完全正面對你，眼睛與方塊等高
```

### X 軸 = 水平旋轉（左右）= 「被吃了多少」

```
0   = 正面朝你（沒被吃）
+   = 往右轉（露出右側面）
-   = 往左轉（露出左側面）
+1 / -1 = 完全側面（被吃完）
```

### Y 軸 = 垂直旋轉（上下）= 「上下看多少」

```
0   = 平視（眼睛與方塊等高）
+   = 從上面看（看見頂面）
-   = 從下面看（看見底面）
+1 / -1 = 完全從正上方/正下方看
```

### 一句話記法

```
X = 被吃了多少
Y = 上下看多少
```

### Z 軸 = 深度（之後處理）

```
現在不處理
之後用來算：物體離你有多遠 → 縮放比例
```

------

## 二、數值刻度

```
-1  ←  -3/4  ←  -2/4  ←  -1/4  ←  0  →  1/4  →  2/4  →  3/4  →  1
```

| 數值 | 物理含義          | 看到幾個面       |
| ---- | ----------------- | ---------------- |
| 0    | 完全正面/平視     | 1個面            |
| 1/4  | 開始露出側面/頂底 | 開始露出第二個面 |
| 2/4  | 兩個面各半        | 兩個面等分       |
| 3/4  | 快到側面了        | 第二個面占主導   |
| 1    | 完全側面/正上下   | 1個面（另一個）  |

正負號的意義：

```
X軸：+ 右  -  左
Y軸：+ 上看  - 下看
```

360度的對稱規則：

```
你只需要畫 0 到 1 這一段
剩下全都是翻轉：
  左右翻 = X的正負互換
  上下翻 = Y的正負互換
```

------

## 三、方塊召喚法

任何方塊 = (X, Y)

### 例子

| 座標         | 長什麼樣                         |
| ------------ | -------------------------------- |
| (0, 0)       | 正方形（正面對你）               |
| (2/4, 0)     | 平視，左右兩面各半               |
| (1, 0)       | 平視，完全側面（長方形）         |
| (0, 1)       | 從正上方看（正方形，但看見頂面） |
| (0, 2/4)     | 從斜上方看，頂面和正面各半       |
| (2/4, 2/4)   | 從斜上方看，三個面都露出來       |
| (1/4, 1/4)   | 微微偏右、微微從上面看           |
| (-3/4, -1/4) | 幾乎側面，從略下方看             |

------

## 四、三種顏色面的出現規則

彩色圖（藍/綠/橘）：

```
藍色（頂面）：Y > 0 時出現，Y越大藍色越多
綠色（左側面）：X < 0 時出現
橘色（右側面）：X > 0 時出現
```

哪個面多哪個面少：

```
數值越接近 1 或 -1 → 對應的面越多
數值越接近 0 → 正面越多
```

------

## 五、月食類比（補充理解用）

如果數值還是很難直覺，可以用這個辅助圖像：

```
0    = 滿月（完全正面）
1/4  = 開始被吃
2/4  = 半月（兩面各半）
3/4  = 月牙（快被吃完）
1    = 新月（完全側面）
```

水平方向和垂直方向各自獨立地「吃」。

但長期來看直接用數值比月食更快，因為數值可以直接運算。

------

## 六、快速參考圖（文字版）

```
        Y = -1（從正下方看）
           |
        Y = -3/4
           |
        Y = -2/4
           |
        Y = -1/4
           |
X=-1 — X=-3/4 — X=-2/4 — X=-1/4 — (0,0) — X=1/4 — X=2/4 — X=3/4 — X=1
           |
        Y = 1/4
           |
        Y = 2/4
           |
        Y = 3/4
           |
        Y = 1（從正上方看）
```

中心 (0,0) = 錨點。往哪個方向走，就是該方向的月食開始進行。
